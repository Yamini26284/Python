# My CS50p Python Journey

This repository documents my progress and solutions for the problems presented in Harvard University's CS50's Introduction to Programming with Python (CS50p) course.

## About CS50p

CS50p is an introductory course that teaches students how to program with Python. It covers fundamental concepts of programming, including functions, variables, conditional statements, loops, data structures, and more.

* **Course Link:** [https://cs50.harvard.edu/python/2022/](https://cs50.harvard.edu/python/2022/) (You can update the year if a new version is released)
